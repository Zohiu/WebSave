WebSave - Website downloader

Save websites in a local directory to access them offline.
Available as a progressive web app in supported browsers/devices.

Before you can use this, you have to select a directory.

Warning: Your browser does not support direct folder access!

All downloaded websites will be stored in local browser storage instead of your filesystem, which means you won't be able to access the html files directly and can't easily transfer data between devices. Please consider using Chrome, Edge or Opera!

If the website has been saved before, it might be overwritten!
